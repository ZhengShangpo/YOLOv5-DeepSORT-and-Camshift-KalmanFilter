# YOLOv5-DeepSORT-and-Camshift-KalmanFilter
这个项目主要涉及到了基于深度学习的目标跟踪算法YOLOV5+DeepSORT与基于传统的单目标跟踪算法Camshift+Kalman滤波，本项目将两种算法集成到软件当中，通过执行main_gui.py文件，即可运行该项目。
