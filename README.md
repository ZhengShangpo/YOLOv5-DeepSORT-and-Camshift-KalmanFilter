# YOLOv5-DeepSORT-and-Camshift-KalmanFilter
这个项目主要用到了基于深度学习的目标跟踪算法YOLOV5+DeepSORT与基于传统的单目标跟踪算法Camshift+Kalman滤波，本项目将两种算法集成到软件当中，运行main_gui.py，即可运行该项目。
